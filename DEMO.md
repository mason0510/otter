## 🎙️ 演示脚本（逐字稿）

### 开场（20 秒）

```
【画面：浏览器打开 http://82.29.54.80:3025】

大家好，我是 Sui Intent Agent，一个基于 AI 的 Sui 交易意图解析器。

我们的核心创新是：让用户用自然语言描述交易意图，
AI 自动解析并生成可验证的 PTB（Programmable Transaction Blocks）。
```

### 场景 1: 简单 Swap（45 秒）

```
【画面：在输入框输入 "把 10 SUI 换成 USDT，滑点 3%"】

首先，看一个简单的代币兑换场景。

用户只需要输入："把 10 SUI 换成 USDT，滑点 3%"

点击"解析意图"...

【画面：右侧显示交易摘要】

AI 成功解析出：
- 操作类型：Swap
- 输入代币：10 SUI
- 输出代币：USDT
- 滑点：3%
- 置信度：95%

更重要的是，系统展示了：
- 预计 Gas 费用
- 交易步骤数
- 风险提示（价格波动）

这就是我们强调的"可验证"——用户在签名前，
能完全理解交易内容和潜在风险。
```

### 场景 2: 复合操作（60 秒）

```
【画面：清空输入，输入 "先换 20 SUI 成 USDC，然后转一半给 Alice"】

接下来展示更复杂的场景——复合操作。

用户输入："先换 20 SUI 成 USDC，然后转一半给 Alice"

【画面：右侧显示 2 个操作】

AI 识别出两个意图：
1. Swap: 20 SUI → USDC
2. Transfer: 10 USDC → Alice

关键点来了——这两个操作会打包成**一个 PTB**，
实现原子执行：要么全部成功，要么全部失败。

这展示了 Sui PTB 的核心优势：
- 一步签名完成多步操作
- 减少中间状态风险
- 节省 Gas 费用

系统自动计算：
- 总步骤：2 步
- 预计 Gas：0.01 SUI
- 风险：转账不可撤销
```

### 场景 3: 安全验证（45 秒）

```
【画面：输入 "把 10000 SUI 换成 USDT"】

安全是我们的核心关注点。

如果用户输入："把 10000 SUI 换成 USDT"

【画面：显示错误提示】

系统会拦截：
"金额无效或超出限制：10000"

因为我们设置了策略限制：
- 单笔交易上限：1000 SUI
- 滑点上限：5%
- Token 白名单：SUI/USDT/USDC

这确保了即使 AI 解析错误，
也不会造成超出用户预期的损失。
```

### 场景 4: Split 拆分（30 秒）

```
【画面：输入 "把我的 SUI 平均分成 3 份"】

最后一个场景——余额拆分。

用户输入："把我的 SUI 平均分成 3 份"

【画面：显示拆分摘要】

AI 自动计算比例：
- 33.33%, 33.33%, 33.34%

确保总和为 100%，避免资金丢失。
```

### 总结（30 秒）

```
【画面：返回首页，展示整体界面】

总结一下 Sui Intent Agent 的核心价值：

1. **自然语言交互**
   - 降低 Web3 使用门槛
   - 无需理解复杂交易参数

2. **可验证的交易**
   - 人类可读的交易摘要
   - 明确的风险提示

3. **策略引导的安全**
   - Token 白名单
   - 金额和滑点限制
   - 参数自动校验

4. **AI + Sui PTB**
   - 多步操作原子执行
   - 一次签名完成复杂交易

我们的愿景是：让每个人都能安全、便捷地使用 Sui 生态。

谢谢！
```

1. **浏览器设置**

1. **鼠标移动**
   - 缓慢、平滑

   - 输入框（打字时）
   - "解析意图"按钮（点击时）
   - 右侧摘要（结果展示时）

## 🚀 快速开始

### 本地测试

```bash
# 1. 确保本地服务运行
cd /Users/houzi/code/06-production-business-money-live/website-fetch-analysize-data-skills/sui-intent-agent
npm run dev

# 2. 打开浏览器
open http://localhost:3000

# 3. 测试场景
python3 -c "
import requests

cases = [
    '把 10 SUI 换成 USDT，滑点 3%',
    '先换 20 SUI 成 USDC，然后转一半给 Alice',
    '把 10000 SUI 换成 USDT',
    '把我的 SUI 平均分成 3 份'
]

for msg in cases:
    print(f'\\n输入: {msg}')
    r = requests.post('http://localhost:3000/api/intent',
        json={'message': msg})
    print(r.json())
"
```

   - 或后期配音
